Documentation
 
 

 

The MonALISA Service system is entirely written in Java (requires J2SE 1.4.2) and is using the following external packages:
JINI

http://www.jini.org
GLUE (web services)

http://www.themindelectric.com
AXIS (web services)

http://ws.apache.org/axis/
JOESNMP (simple SNMP client)

http://www.opennms.org/
Apache Tomcat (servlet container)

http://jakarta.apache.org/tomcat/
McKoi (Embedded SQL DB in Java)

http://mckoi.com/database/
MySQL (MySQL DataBase)

http://mckoi.com/database/
Java3D (optional for GUIs)

http://java.sun.com/products/java-media/3D/

For data storage the system is using JDBC. The default data base is McKoi DB, but this may be changed with runtime options to connected to virtually any relational DB for which jdbc driver is provided. McKoi is a simple DB system, written in java, which is embedded in this monitoring framework and makes the deployment of this package very easy.

The Graphical Clients for services are available as Web Start applications. This allows to start and install them easily from any browser.

Perl clients for the Web Services need to install SOAP::Lite for Perl (http://www.soaplite.com)

Java clients for the Web Services may use WSIF (http://ws.apache.org/wsif)
