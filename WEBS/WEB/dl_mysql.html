MySQL for ML
 
 

 

An easy to use package for installing MySQL to be used in MonALISA

If you would like to use MySQL to store the monitoring values this archive will allow you to easily install and create the tables MonaLisa needs. To install MySQL data base does not require root privileges. MySqlCC is also included. They may be used to see the collected values. The archive contains the binary distribution for Linux (i386 platform libc6) version 4.0.13 with InnoDB support for transactions. For more details visit: www.mysql.com and www.innodb.com.
