Tests and examples of using MonALISA

MonALISA has its own monitoring module witch allows to see the resources the program is using and at the same time collects performance numbers. We used this functionality to evaluate the way MonALISA works with really large farms and high rate of data acquisition.

The "Large Farm" tests were done at CERN, using a 500 nodes cluster (lxbatch)
