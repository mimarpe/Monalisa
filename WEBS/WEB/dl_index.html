MonALISA Download
version 1.0.5, October 25, 2003

To install and run the MonaLisa Service please download the service distribution and follow the instructions from the User Guide and the README files.

The distribution Contains:
  • MonaLisa Service
    • SNMP support
    • Embedded data base & support for MySql & PostgreSQL
    • SSL Administrative interfaces
    • Interface modules with Ganglia, LSF, PBS, Condor-Hawkeye
    • Examples to write Monitoring Modules or interfaces to other tools
    • Customized additional data writers
    • WSDL/SOAP clients
  • Client GUI
  • Documentation
It requires J2SE 1.4.2
or later!
The most convenient way to start a global GUI for the MonALISA is by using the Java Web Start technology which will install the application on your system using the browser.
